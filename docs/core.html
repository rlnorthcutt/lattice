Lattice Core

The foundation layer of utilities that power layout, spacing, and typography across every Lattice project.

Installation

Include the compiled stylesheet from npm, a CDN, or the repository. The Core layer is intentionally small and tree-shakeable if you use the source files.

# npm
npm install @lattice-css/core

# import into your CSS build
@import "@lattice-css/core";

# or drop the bundle in HTML
<link rel="stylesheet" href="https://unpkg.com/@lattice-css/core/dist/lattice-core.min.css">

Philosophy

  • Predictable scales — spacing, sizing, and typography utilities share a consistent numeric rhythm.
  • Accessibility first — defaults embrace accessible contrast, focus rings, and motion sensitivity.
  • Composition over components — keep markup expressive by combining small, purposeful classes.

Key Utility Families

Category Prefix Example
Spacing p-, m- p-4 applies uniform padding from the core spacing scale.
Layout flex-, grid- flex-center creates a flex container centered on both axes.
Typography text-, font- text-lg increases type size while keeping the rhythm intact.

Example

Combine utilities to ship production-ready interfaces without leaving HTML.

<section class="p-6 bg-surface rounded-lg shadow-soft">
  <h2 class="text-xl font-semibold mb-2">Create once, reuse everywhere</h2>
  <p class="text-base text-muted">
    Core utilities let you iterate quickly and stay consistent across features.
  </p>
  <div class="mt-4 flex gap-3">
    <button class="btn btn-primary">Primary action</button>
    <button class="btn btn-ghost">Secondary</button>
  </div>
</section>
