Lattice.css

A layout-first utility layer: predictable grids, tiny helpers, and zero opinions about your typography or components.

Why Lattice?

Lattice focuses on the “boring math” of layout so your team can keep typography, colors, and components in your own design system. You get:

  • Master 12-col page grid with fluid outer gutters.
  • Component grids that default to 12 equal tracks for simple alignment.
  • One-class responsive columns (.grid .col-N) for equal layouts without extra wrappers.
  • Tiny utility set for visibility/positioning. Need more? Add Extra for spacing, flex helpers, and unequal spans.

Strategy

Keep Core Tiny

Ship only the 12-col page grid, equal-column helpers, and minimal utilities. Fewer classes, faster loads, easier maintenance.

Explore Core

Opt Into Extras

Add spacing scales, display/flex helpers, and unequal spans only when you need them—your bundle stays lean by default.

Own Your Aesthetics

Pair Lattice with a classless base like Ivy for sensible defaults, while your design system controls typography, color, and components.

Getting Started

CDN (quickest path)

Drop these tags in your <head>:

<!-- Ivy (classless base) -->
<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/rlnorthcutt/ivy/dist/ivy.full.min.css">

<!-- Lattice (full bundle) -->
<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/rlnorthcutt/lattice/dist/lattice.full.min.css">

Local bundle (recommended for this docs site)

Your workflow copies the minified Lattice full build into docs/:

<link rel="stylesheet" href="./lattice.full.min.css">
What’s in “Core” vs “Extra”?
  • Core: 12-col page grid, .grid with 12 equal tracks, equal-column helpers (.col-1,2,4,6,8,12), responsive sm/md/lg/xl/xxl-col-*, minimal display/visibility.
  • Extra: spacing/gap scale, display toggles, flex/positioning helpers, and the remaining spans (.span-1..12 + responsive) for unequal layouts.

Examples

Equal Columns (one class per breakpoint)

On small screens items stack; at larger breakpoints they split evenly.

<section class="grid sm-col-2 lg-col-3 gap-2">
  <div class="card"><h4>Tile 1</h4><p>Auto stack → 2 → 3 columns.</p></div>
  <div class="card"><h4>Tile 2</h4><p>No extra wrappers needed.</p></div>
  <div class="card"><h4>Tile 3</h4><p>Equal widths by track count.</p></div>
</section>

Unequal Columns (add Extra)

Use child spans when you need 25/75, 3/9, etc.

<section class="grid gap-2">
  <aside class="span-3 card"><h4>Sidebar (3/12)</h4></aside>
  <main class="span-9 card"><h4>Content (9/12)</h4></main>
</section>

Full-bleed Section + Centered Content

<header class="full-width">
