Lattice CSS

A modern, utility-first CSS toolkit designed to help you compose clean interfaces with confidence.

Why Lattice?

  • Composable utility classes make it easy to design directly in your markup.
  • Zero JavaScript footprint keeps your pages fast and accessible.
  • Core and Extra modules let you scale from minimal utilities to expressive interfaces.

Getting Started

Drop the stylesheet into your page and begin applying utility classes. The project ships with an expressive Core layer and an optional Extra layer for extended patterns.

<link rel="stylesheet" href="https://unpkg.com/@lattice-css/core/dist/lattice-core.min.css">
<link rel="stylesheet" href="https://unpkg.com/@lattice-css/extra/dist/lattice-extra.min.css">
