Lattice CSS

A lightweight, flexible CSS utility library for modern web development

What is Lattice?

Lattice is a lightweight CSS utility library that provides essential building blocks for creating beautiful, responsive web interfaces. Inspired by modern utility-first frameworks, Lattice gives you the power to rapidly build custom designs without leaving your HTML.

Quick Start

Getting started with Lattice is easy. Simply include the CSS file in your HTML:

<link rel="stylesheet" href="lattice.min.css">

Or use the CDN (coming soon):

<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/rlnorthcutt/lattice/lattice.min.css">

Then start using utility classes in your HTML:

<div class="flex items-center justify-center p-4">
  <h1 class="text-2xl font-bold text-gray-900">Hello, Lattice!</h1>
</div>
