Lattice Extra

Optional enhancements that layer on top of Core utilities to deliver polished, production-ready UI patterns.

Installation

The Extra bundle is published separately so that you only ship what you need.

# npm
npm install @lattice-css/extra

# import alongside the core layer
@import "@lattice-css/extra";

# via CDN
<link rel="stylesheet" href="https://unpkg.com/@lattice-css/extra/dist/lattice-extra.min.css">

When to use Extra

Reach for Extra when you need interactive affordances, refined component primitives, or design tokens that extend the base theme.

  • Component primitives such as buttons, badges, cards, and alerts.
  • Design tokens for motion, shadows, and gradients that match the Lattice aesthetic.
  • State-driven utilities that respond to hover, focus-visible, and data attributes.

Featured Patterns

  • Cards

    Compose card, card-header, and card-body utilities to build responsive card layouts.

    <article class="card shadow-soft">
  <header class="card-header flex-between">
    <h3 class="text-lg">Release notes</h3>
    <span class="badge badge-soft">v2.1</span>
  </header>
  <div class="card-body text-muted">
    Meet the latest refinements to typography and spacing scales.
  </div>
</article>

  • Command palette

    Pair surface-elevated, backdrop-blur, and key bindings classes for a delightful command palette.

    <div class="palette surface-elevated backdrop-blur-md" role="dialog">
  <input class="palette-input" placeholder="Search commands" autofocus>
  <ul class="palette-list" role="listbox">
    <li class="palette-item is-active">Open docs</li>
    <li class="palette-item">Toggle theme</li>
  </ul>
</div>
